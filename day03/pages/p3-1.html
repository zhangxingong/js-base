闭包

  • 常见的创建方式在函数内部创建另一个函数
  • 闭包的第一个用途：通过闭包可以访问局部变量
  • 闭包的第二个用途：可以让这些变量的值始终保持在内存中
  • 有点：可以把局部变量驻留内存中，可以避免使用全局变量
每天解决一个bug，进步一点点。。。
            //闭包
            //----------START--------------------
                function myfn() {
                    return function (){
                        alert("........");
                    }
                }
                myfn()();
            //-------------END---------------------
                function fn1(){
                    var local = "bb";
                    return function (){
                        return local;
                    }
                }
                alert(fn1()());

                var tampfn = fn1();
                alert(tampfn());
            //--------------END------------------
